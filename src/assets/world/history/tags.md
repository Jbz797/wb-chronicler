# 🏷 Vocabulaire des tags

<p class="metadata">Date de mise à jour : 01/05/26 13:11</p>

| Tag | Description |
|-----|-------------|
| `CATACLYSM` | Événement géologique ou magique majeur (éruption, raz-de-marée, tempête, etc.) |
| `EXTINCTION` | Extinction d'une espèce intelligente |
| `FAVORITE-DEATH` | Le favori suivi est mort dans ce chapitre (le successeur est désigné dans le même chapitre) |
| `FIRST-CLAN` | Apparition du premier clan |
| `FIRST-FAVORITE` | Désignation du tout premier favori |
| `FIRST-KINGDOM` | Premier royaume du monde |
| `FIRST-RELIGION` | Apparition de la première religion |
| `FIRST-VILLAGE` | Fondation du premier village du favori |
| `FIRST-WAR` | Première déclaration de guerre du monde |
| `GENESIS` | Premier chapitre du monde |
| `NOTABLE-ENCOUNTER` | Rencontre marquante du favori avec une figure d'importance |
