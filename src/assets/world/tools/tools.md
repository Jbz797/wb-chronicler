# 🛠 Outils du chroniqueur

<p class="metadata">Date de mise à jour : 13/05/26 02:20</p>

| Script | Usage | Sortie |
|--------|-------|--------|
| `creature-traits/lookup.py <id> [<id>...]` | Résout un ou plusieurs traits de créature. | Une ligne par trait : `id \| rareté \| description \| flavor \| stats` |
| `equipment/lookup.py <chapterIndex\|current> <id> [<id>...]` | Résout un ou plusieurs items d'équipement. | Une ligne par item : `id \| rareté \| asset_id \| durabilité \| by \| from \| kills \| âge \| stats` |
| `genetics/stats.py <id>` | Résout les gènes d'une sous-espèce. | `id \| stats` |
